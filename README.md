## Hospital Management System

Sistema de gerenciamento hospitalar desenvolvido em Python com integração ao MySQL.

### Objetivo

Este projeto tem como objetivo praticar conceitos de desenvolvimento backend, banco de dados e regras de negócio através da criação de um sistema de gestão hospitalar.

O sistema foi originalmente concebido durante o curso Programador Backend e Dados do SENAI e está sendo reconstruído e aprimorado para fins de estudo e aplicação de boas práticas de desenvolvimento.

### Funcionalidades

* Cadastro de pacientes
* Cadastro de médicos
* Cadastro de consultas
* Edição de registros
* Exclusão de registros
* Relatórios gerenciais
* Controle de status de médicos
* Consultas filtradas por critérios específicos

### Tecnologias

* Python
* MySQL
* Git & GitHub
